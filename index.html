Title

Git GUI - это кроссплатформенный и переносимый интерфейс на основе Tcl/Tk для Git, который работает в Linux, Windows и Mac OS X. Он в основном фокусируется на генерации коммитов, позволяя пользователям вносить изменения в свой репозиторий путем генерации новых коммитов, внесением изменений в существующие, строительством филиалов.
